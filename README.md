# SonMAT

CS408(Computer Science Project) in Fall2016.
Online mathmetical handwriting IME service


## install

```/bin/sh
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python manage.py initdb
```

## run
```/bin/sh
$ source venv/bin/activate
$ python manage.py runserver
```
